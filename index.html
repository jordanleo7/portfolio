FreeCodeCamp Projects

Simon Game

This game presents users with a series of button presses with corresponding sounds. The user replicates the pattern and if correct, the game repeats the series with one more button added. If the user presses the wrong button, then the series is repeated and the user can try again. The strict button disables trying again after a mistake and the game simply starts over.

Tic Tac Toe

Pomodoro Clock

A pomodoro clock is two timers that take turns counting down for as long as the user needs. For example, after a 25 minute work time elapses, a 5 minute break time begins. After the 5 minutes elapse, the 25 minute timer starts again, and so on. Both timer lengths are customizable.

JavaScript Calculator

Twitch.tv JSON API

This site uses the Twitch.tv API to check if a list of user accounts are online, offline, or deactivated, and if they are online, what game they are playing, and their stream title. This information is then displayed to the user.

Wikipedia Viewer

Using Wikipedia's API, the user can search for Wikipedia entries, or click a button to see a random entry.

FreeCodeCamp Projects

Simon Game

A memory game that creates a series of button presses for the player to repeat. If the player is successful, the series is repeated with an additional step. If the player fails then the series is repeated, but if the strict button is on then the game ends.

Tic Tac Toe

Play tic tac toe against a friend or computer. The computer opponent is controlled by JavaScript. The easy computer opponent has less code, so its moves are more random. The hard opponent has more code, and is challenging but not impossible to beat.

Pomodoro Clock

A pomodoro clock has two timers that continuously take turns counting down. For example, the user could set the first timer to 25 minutes for work, and the second timer for 5 minute breaks. When one timer ends, the other begins, until the user clicks stop.

JavaScript Calculator

A simple calculator that allows chaining operations together.

Twitch.tv JSON API

Display a list of account statuses using Twitch.tv's API. If online, show their stream title and game.

Wikipedia Viewer
